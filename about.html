About Me And This Website

Ciao! My name is Andrea Di Fonzo, I'm Italian and I've been based in Geneva, Switzerland, since 2011.
I've four main passions: paragliding, landscape photography, coding and cooking fantastic homemade pizzas!

All the pictures in this website have been taken by me.

I've always had the dream to build my website, so in January 2020 I've finally decided to combine two of my passions: the result is this website!

I used Mimo, a great app that helped me both with the HTML, CSS and JAVA theory, and most importantly, to build the habit to code every day. It took me around three months to complete this project, investing around 30min per day.

You can achieve the same result in a couple of days with specific apps, like Wix, but my objective was to learn to code, so this way was more effective and fun!

If you are interested in usage rights for any of my pictures or you simply want to get in touch, email me!
