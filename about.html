About Me And This Website

Ciao! My name is Andrea Di Fonzo, I'm originally from Italy and I've been based in Geneva since 2011.
I've four main passions: paragliding, landscape photography, coding and prepare fantastic homemade pizzas!

All the pictures you can see in this website have been taken by me.

I've always had the dream to build my website, so in January 2020 I've finally decided to combine two passions: coding and photography. The result is this website!

Mimo was the great app that helped me both to learn coding in HTML, CSS and JAVA, and, most importantly, to build the habit to code every day. It took me around three months to complete this project, investing around 30min per day.

You can achieve the same result probably in just one day with other apps, like Wix, but my objective was to learn to code so I found that this way more fun! :D

If you are interested in usage rights for any of my pictures or you simply want to get in touch, email me!
